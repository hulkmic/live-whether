
# Project Overview

This is a Flutter application that provides live weather information. Users can search for a city, and the app will display the current weather conditions for that location.

# Style, Design, and Features

*   **UI:** The app will have a clean and simple user interface with a search bar and a section to display weather details.
*   **State Management:** The app will use the `provider` package for state management.
*   **API Integration:** The app will fetch weather data from a third-party API.
*   **Theming:** The app will have a consistent theme with a defined color scheme and typography.

# Current Request: Build the Weather App

## Plan:

1.  **Add Dependencies:** Add `http` for API calls and `provider` for state management.
2.  **Create UI:** Design the main screen with a search field, a search button, and a display area for weather data.
3.  **Weather Service:** Implement a service to fetch weather data from an API.
4.  **State Management:** Create a `WeatherProvider` to manage the app's state, including the weather data, loading status, and errors.
5.  **Integrate and Test:** Combine the UI, service, and state management, and ensure the app functions correctly.
